Fatality CS2 Lua Framework

Modern Fatality CS2 Lua Framework.

NXC is a polished Lua framework built for Fatality CS2, focused on clean visuals, useful rage helpers, anti-aim tools, misc utilities, and a modern custom menu experience.

The script is designed to stay clean, compact, and easy to use. Features are off by default, so users can enable only what they want.
